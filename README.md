# star
星座


2015.10.30 
当前版本目前只支持win，Android下的查看星座信息以及分享功能（部分设备不可用），
win下支持protocalbuffer以及zeromq，Android下不支持protobuf以及zeromq。
我们使用宏_WINDOWS来进行宏裁剪，这样我们在win下可以对protobuf和zeromq进行测试。
